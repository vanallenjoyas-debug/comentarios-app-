Política de Privacidad

Última actualización: Abril 2026

1. Información general

Esta aplicación es una herramienta interna de gestión de comentarios desarrollada y utilizada exclusivamente por Javier Romero (Joyería Sudaca / Van Allen) para administrar comentarios en sus redes sociales.

2. Datos que se acceden

La aplicación accede a comentarios públicos de las páginas de Facebook administradas por el usuario autorizado. No se recopilan, almacenan ni comparten datos de terceros.

3. Uso de los datos

Los datos accedidos se utilizan exclusivamente para:

  • Visualizar comentarios recibidos en las páginas de Facebook
  • Permitir al administrador responder comentarios desde la aplicación

4. Almacenamiento

La aplicación no almacena permanentemente ningún dato de usuarios de Facebook. Los tokens de acceso se almacenan localmente en el navegador del administrador y se usan únicamente para autenticación con la API de Facebook.

5. Terceros

La aplicación utiliza la API de Facebook (Meta) para acceder a los datos. El uso de estos datos está sujeto a la Política de datos de Meta.
